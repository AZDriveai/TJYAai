## 🌟 نظرة عامة

TJYAai هو تطبيق ويب متطور يستخدم أحدث تقنيات الذكاء الاصطناعي لتحرير وتحويل الصور. يوفر التطبيق مجموعة شاملة من الأدوات المتقدمة لتحسين الصور وتحويلها إلى أنماط فنية مختلفة، بالإضافة إلى ميزات فريدة مثل تحريك الصور للرقص.

## 🎯 الميزات الرئيسية

### 🎨 تحويل الصور بالذكاء الاصطناعي
- ✅ تحويل فوري إلى أنماط متعددة (مانجا، أنمي، كرتون ثلاثي الأبعاد، واقعي)
- ✅ معالجة بنقرة واحدة بدون علامة مائية
- ✅ دعم أكثر من 20 نمط فني مختلف
- ✅ تخصيص المعاملات والإعدادات المتقدمة

### 💃 تحريك الصور للرقص
- ✅ تحويل الصور الثابتة إلى فيديوهات راقصة
- ✅ مكتبة واسعة من قوالب الرقص والحركة
- ✅ تحسين تلقائي للوجوه والحركات
- ✅ إنتاج فيديوهات عالية الجودة (HD/4K)

### 🖼️ معرض شخصي متقدم
- ✅ رفع مباشر للصور مع سحب وإفلات
- ✅ تنظيم في ألبومات مخصصة
- ✅ معاينة سريعة وإدارة متقدمة
- ✅ مشاركة سهلة عبر وسائل التواصل

### 🔧 أدوات تحسين الصور
- ✅ **إصلاح السيلفي (Retake)**: تحسين الصور الشخصية تلقائياً
- ✅ **استعادة الوجوه**: إصلاح الوجوه المشوهة أو غير الواضحة
- ✅ **تكبير الجودة**: زيادة دقة الصور حتى 4x
- ✅ **إزالة الخلفية**: إزالة دقيقة للخلفيات
- ✅ **تغيير تسريحة الشعر**: تجربة تسريحات مختلفة
- ✅ **تغيير الملابس**: تجربة أزياء متنوعة
- ✅ **مكياج افتراضي**: تطبيق مكياج رقمي احترافي

### ✨ ميزات إضافية مبتكرة
- 👶 **AI Baby**: مولد الطفل المستقبلي للأزواج
- 🎮 **مؤثرات اللعبة**: تأثيرات PS2 والألعاب الكلاسيكية
- 🌈 **فلترات زمانية**: تأثيرات 2000s dreamy وعصور مختلفة
- 🎭 **تأثيرات بصرية**: مجموعة واسعة من الفلاتر الفنية

## 🚀 البدء السريع

### المتطلبات الأساسية
- 🐳 Docker & Docker Compose
- 📦 Git
- 🟢 Node.js 18+ (للتطوير المحلي)
- 🐍 Python 3.11+ (للتطوير المحلي)

### التثبيت السريع

```bash
# 1. استنساخ المشروع
git clone https://github.com/AZDriveai/TJYAai.git
cd TJYAai

# 2. إعداد متغيرات البيئة
cp .env.example .env
# قم بتحرير .env وإضافة مفاتيح API الخاصة بك

# 3. تشغيل سكريبت الإعداد التلقائي
chmod +x scripts/setup.sh
./scripts/setup.sh

# 4. تشغيل التطبيق
docker-compose up -d
```

## 🔧 التطوير

### إعداد بيئة التطوير

```bash
# تشغيل الواجهة الأمامية
cd frontend
npm install
npm run dev

# تشغيل الخادم الخلفي
cd backend
python -m venv venv
source venv/bin/activate  # Linux/Mac
pip install -r requirements.txt
uvicorn app.main:app --reload

# تشغيل Celery Workers
celery -A app.celery_app worker --loglevel=info
```

### 🧪 تشغيل الاختبارات

```bash
# اختبارات الخادم الخلفي
cd backend
pytest tests/ -v

# اختبارات الواجهة الأمامية
cd frontend
npm run test

# اختبارات التكامل
docker-compose -f docker-compose.test.yml up --abort-on-container-exit
```

## 🚀 النشر

### النشر للإنتاج

```bash
# بناء صور الإنتاج
docker-compose -f docker-compose.prod.yml build

# تشغيل الإنتاج
docker-compose -f docker-compose.prod.yml up -d

# مراقبة السجلات
docker-compose -f docker-compose.prod.yml logs -f
```

### النشر على Kubernetes

```bash
# تطبيق تكوينات Kubernetes
kubectl apply -f k8s/

# مراقبة حالة النشر
kubectl get pods -n tjyaai
```
